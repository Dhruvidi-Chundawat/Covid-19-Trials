This project applies EDA on the global COVID-19 clinical trials data obtained from ClinicalTrials.gov.
The data analysis is done on trial status, enrollment trends, country-wise contributions,
missing data, and temporal trends.

## Tools
- Python
- Pandas, NumPy
- Matplotlib, Seaborn

## Key Insights
- The majority of the trials are in the United States
- The enrollment data is highly skewed
- A substantial rise in trials in the early phases of the pandemic
